# Запуск проекта

Не забудьте вставить свои данные в `.env` файл. Пример окружений можно найти в файле `.env.example`.

Для запуска проекта используйте следующую команду:


```bash
docker compose up
```

После запуска перейдите в документацию по 0.0.0.0/docs

## Стек технологий

- **FastAPI**
- **SQLAlchemy**
- **Pydantic**
- **Alembic**
- **PostgreSQL**
- **HTTPX**
